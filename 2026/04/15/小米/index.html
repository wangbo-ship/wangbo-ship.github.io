小米技术岗位学习路线

发表于2026-04-15|更新于2026-04-17|求职
|浏览量:

目标岗位：小米武汉 / 北京 → C++ 端侧推理 / 三维重建 / 大模型工程 (Python)

核心优势：导师方向覆盖大模型、高性能计算、三维重建，完美匹配小米技术栈。

一、方向总览与岗位匹配

技术方向 核心岗位 对应小米业务线 校招薪资范围（硕士）
C++ 高性能方向 端侧大模型推理引擎开发、三维重建算法、高性能计算工程师 小米汽车、AI 实验室、AIoT 31-36K × 14 薪（普通）40K+ × 16 薪（SP/SSP）
Python 大模型方向 大模型应用开发、RAG 系统工程师、Agent 架构师 小爱同学、大模型平台、云服务 33-39K × 14-16 薪

二、C++ 方向学习路线（6 个月攻坚版）

阶段目标：打造底层系统能力，攻克端侧推理与三维重建工程化。

阶段 周期 核心知识点 掌握标准 推荐实战项目
1. 基础硬核 第 1 个月 C++11/20 深度（智能指针、多线程、模板元编程）
Linux 系统编程（IO 多路复用 Epoll、进程间通信、网络编程）		 能独立手写基于 Epoll 的高性能 HTTP 服务器，无内存泄漏 手写高性能内存池
实现简单 Reactor 网络模型
2. 性能优化 第 2 个月 GCC 编译选项（-O2/-g）、GDB/Valgrind 调试
汇编与底层（寄存器、指令级优化 ILP、SIMD 指令加速）
并行计算（OpenMP、CUDA 基础）		 理解 CPU 缓存命中机制，能定位并优化 C++ 代码性能瓶颈 对矩阵运算或点云处理算法进行 SIMD 加速优化
3. 端侧大模型 第 3-4 个月 推理框架（MNN、TensorRT Lite、ONNX Runtime）
大模型推理（FlashAttention 原理、KV Cache 量化与管理）
部署实战（C++ 前端调用大模型推理接口）		 成功跑通小模型（如 Qwen-7B）的 C++ 端侧推理 Demo 基于 MNN 框架实现 7B 模型本地离线推理
4. 三维重建 第 5-6 个月 计算机视觉（OpenCV 特征提取、ORB 特征、PnP 算法）
三维图形学（点云处理 PCL、3DGS）
BA 优化（光束法平差、高斯牛顿法）		 能从 2D 图片重建 3D 点云模型，完成 C++ 系统封装 基于 OpenCV+PCL 实现小规模三维场景重建系统

三、Python 大模型方向学习路线（4 个月速成版）

阶段目标：快速构建企业级大模型应用能力，掌握 RAG 与 Agent 开发。

阶段 周期 核心知识点 掌握标准 推荐实战项目
1. 工程基石 第 1 个月 Python 高级（asyncio 异步、FastAPI、类型提示）
数据处理（NumPy、Pandas、向量数据库 Milvus/Chroma）		 能独立开发高并发 RESTful API 服务 设计企业级 RAG 接口服务，支持多文档异步上传
2. 大模型基础 第 2 个月 Transformer 架构（Encoder/Decoder、Attention）
模型微调（LoRA/QLoRA、数据集构建）
推理加速（vLLM、SGLang、TensorRT-LLM）		 能解释大模型推理加速核心原理，调用主流模型 API 基于 vLLM 搭建本地大模型推理服务，模拟高并发
3. RAG 实战 第 3 个月 检索增强（文档切分 Sentence-BERT、混合检索）
重排策略（Rerank 模型、上下文压缩）
全链路开发（LangChain/LlamaIndex）		 搭建支持多模态检索的问答系统，准确率 80%+ 开发基于本地知识库的智能问答机器人（支持 PDF/Word/图片）
4. Agent 开发 第 4 个月 工具调用（Function Call、Tool Calling）
多智能体（任务规划、反思机制、多轮对话管理）
部署上线（Docker 容器化、微服务架构）		 开发具备工具调用能力（联网搜索、计算等）的智能 Agent 实现 AutoGPT 风格 Agent，自主完成复杂任务规划与执行

四、综合时间规划建议（总耗时约 6 个月）

1. 并行策略（推荐）

  • 前 3 个月：双线并行

    • 工作日/晚间：主攻 C++ 基础与性能优化
    • 周末/碎片时间：学习 Python 大模型基础与 RAG 开发

  • 后 3 个月：深度攻坚

    • 根据小米岗位投递情况，二选一作为核心项目落地（建议选 C++ 端侧 + 大模型，这是小米最强壁垒）

2. 项目落地关键

  • 必须产出 3-5 个可运行的 GitHub 项目
  • C++ 方向重点：端侧大模型推理 Demo、三维重建 C++ 工程
  • Python 方向重点：私有化 RAG 问答系统、多功能 Agent

3. 求职准备

  • 研一/研二暑假：争取进入导师组项目实习，或投递小米暑期实习
  • 校招前：将项目经验转化为 STAR 法则 写进简历，突出 高性能、低延迟、工程落地

五、核心资源清单

书籍

  • 《C++ Primer》
  • 《Unix 环境高级编程》
  • 《深入理解计算机系统》（CSAPP）

框架与库

  • 推理框架：MNN、TensorRT、vLLM
  • 大模型开发：LlamaIndex、LangChain
  • 三维处理：OpenCV、PCL、PyTorch3D

学习网站

  • 算法刷题：LeetCode
  • 视频教程：B站技术 UP 主
  • 开源项目：GitHub
